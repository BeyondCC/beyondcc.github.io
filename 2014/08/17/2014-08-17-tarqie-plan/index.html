从7月17号入职实习到今天已经完完整整一个月了，但是感觉自己是浑浑噩噩度过来得，于是反复思考之后发现是因为没给自己制定计划，没给自己下挑战，so果断地参考网上一篇讲量化成长的文章之后，开始制定自己的计划，开始新的征程

Tarqie

  • 目标
    1. 熟悉iOS基本开发流程
    2. 学完《social psychology》
    3. 坚持锻炼，提高精力
    4. 提升算法
  • 量化
    1. 跟完stanford iOS公开课
    2. 看完最后2个week的社会心理学视频
    3. 看完Princeton算法公开课
  • 改进
    1. 早睡早起（12：00PM ~ 8：00AM）+ 每天跑步4圈
    2. 阅读《社会心理学》中文版第八版
    3. 每天抽一个小时看《Algorithms》，实现对应的数据结构和算法
    4. 每一到两周写一篇blog，对所学知识做一个总结
  • 验证
    1. 早睡早起 + 运动不间断
    2. 学习能力增强，增加心理学知识
    3. 逐步实现基本数据结构和算法
    4. 熟悉iOS基本开发流程，完成mentor分配的任务

总结

在接下来的一个月正式开始实施该计划，just fighting！最后贴一张励志图
